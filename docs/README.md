# Docs

Folder ini berisi dokumentasi teknis untuk pipeline data cleaning lowongan pekerjaan.

## File yang tersedia:

### 1. `data_format.md`
- **Deskripsi**: Penjelasan format standar dataset lowongan pekerjaan
- **Konten**:
  - Struktur kolom dataset
  - Format dan tipe data
  - Standar kategorisasi (level pekerjaan, status, dll)
  - Spesifikasi input dan output

### 2. `pipeline_overview.md`
- **Deskripsi**: Alur pipeline + diagram mermaid
- **Konten**:
  - Flowchart proses pembersihan data
  - Penjelasan setiap tahap pipeline
  - Dependensi antar komponen
  - Diagram arsitektur sistem

## Tujuan Dokumentasi:
- Memberikan pemahaman mendalam tentang pipeline
- Standardisasi format data
- Panduan untuk pengembangan lebih lanjut
- Referensi untuk troubleshooting

## Kontribusi:
- Update dokumentasi setiap ada perubahan pipeline
- Gunakan format Markdown untuk konsistensi
- Tambahkan diagram Mermaid untuk visualisasi yang kompleks
