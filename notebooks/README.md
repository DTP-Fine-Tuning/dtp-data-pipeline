# Notebooks

Folder ini berisi notebook eksplorasi & prototyping untuk analisis data.

## File yang tersedia:

### 1. `data_clean (1).ipynb`
- **Deskripsi**: Notebook untuk eksplorasi dan prototyping proses pembersihan data
- **Fungsi**: 
  - Analisis data mentah
  - Eksperimen dengan berbagai metode cleaning
  - Prototyping sebelum implementasi di script Python
  - Visualisasi data dan hasil cleaning

## Penggunaan:
- Gunakan notebook ini untuk eksperimen dan analisis data sebelum produksi
- Cocok untuk prototyping fitur baru atau debugging
- Ideal untuk eksplorasi data dan visualisasi

## Cara menjalankan:
```bash
# Install Jupyter jika belum ada
pip install jupyter

# Jalankan Jupyter
jupyter notebook

# Atau menggunakan VS Code dengan extension Python/Jupyter
```

## Catatan:
- Notebook bersifat eksploratif, untuk produksi gunakan script di folder `scripts/`
- Pastikan virtual environment sudah diaktifkan
- File notebook dapat berisi output yang sudah dijalankan sebelumnya
